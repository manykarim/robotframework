# Getting Started

Robot Framework is a generic automation framework that works equally well for acceptance testing, ATDD, and RPA — the same readable, keyword-driven syntax everywhere. [Introduction](introduction.md) covers the architecture and core concepts; start there. If you prefer learning by doing, the [demonstration projects](demonstration.md) are a faster on-ramp than any amount of reading.

- [Introduction](introduction.md) — What Robot Framework is, its architecture, and the core concepts behind keyword-driven automation.
- [Demonstrations](demonstration.md) — Demo projects that let you explore Robot Framework examples hands-on.
- [Copyright and License](copyright-and-license.md) — Licensing terms for Robot Framework and its documentation.
